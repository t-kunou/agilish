# Agilish（アジャイル百人一首）

アジャイル開発のプラクティスを、百人一首形式のカードゲームで楽しく学べるWebアプリケーションです。

## 概要

読み上げられるプラクティスの説明文を聞いて、対応するカードを素早く取る競技かるた風のゲームです。スクラム、XP、チームビルディングなど30のアジャイルプラクティスを収録しています。

## 遊び方

1. 難易度を選択してゲームを開始
2. プラクティスの説明文が音声で読み上げられる
3. 該当するプラクティス名のカードをクリック
4. 正解するとカードが取り除かれ、すべてのカードを取ればゲーム終了

### 難易度

| レベル | 枚数 | 制限時間 |
|--------|------|----------|
| EASY   | 10枚 | なし     |
| NORMAL | 20枚 | 30秒/問  |
| HARD   | 30枚 | 15秒/問  |

### スコア

- 正解: +100点 + 残り時間ボーナス（制限時間ありの場合）
- お手つき: -25点
- 時間切れ: -50点

## 収録カテゴリ

- **Scrum** — デイリースクラム、スプリント、スプリントレビューなど
- **Scrum（補完）** — ベロシティ、プランニングポーカーなど
- **XP** — ペアプログラミング、TDD、継続的インテグレーションなど
- **価値探索** — ユーザーストーリーマッピング、リーンキャンバス
- **チームビルディング** — インセプションデッキ

## 起動方法

ビルド不要の静的Webアプリケーションです。ローカルサーバーを起動してブラウザからアクセスしてください。

```bash
python -m http.server 8000
```

`http://localhost:8000/` をブラウザで開きます。

## 技術スタック

- HTML5 / CSS3 / JavaScript（ES6+）
- Web Speech API（音声読み上げ）
- Web Audio API（効果音生成）
- 外部ライブラリ依存なし
