# AGILISH

アジャイル百人一首 — アジャイル開発のプラクティスを、百人一首形式のカードゲームで楽しく学べるWebアプリケーションです。

## 遊び方

1. 難易度を選択してゲームを開始
2. プラクティスの説明文が音声で読み上げられる
3. 該当するプラクティス名のカードをクリック
4. 正解するとカードが取り除かれ、すべてのカードを取ればクリア

## 難易度

| レベル | 枚数 | 制限時間 | ライフ | 読み上げ速度 |
|--------|------|----------|--------|-------------|
| EASY | 10枚 | なし | 無限 | 0.8x |
| NORMAL | 20枚 | 30秒/問 | 10 | 1.0x |
| HARD | 30枚 | 15秒/問 | 5 | 1.2x |
| INFERNO | 40枚 | 12秒/問 | 3 | 2.0x |
| NIGHTMARE | 全札 | 10秒/問 | 1 | 3.0x |

## スコア

- 正解: +100点 + 残り時間ボーナス
- お手つき: -25点
- 時間切れ: -50点

## 収録カテゴリ

- **Scrum** — デイリースクラム、スプリント、スプリントレビューなど
- **Scrum（補完）** — ベロシティ、プランニングポーカーなど
- **XP** — ペアプログラミング、TDD、継続的インテグレーションなど
- **価値探索** — ユーザーストーリーマッピング、リーンキャンバスなど
- **チームビルディング** — インセプションデッキなど

## 起動方法

ビルド不要の静的Webアプリケーションです。

```bash
cd docs
python -m http.server 8000
```

`http://localhost:8000/` をブラウザで開きます。

## 技術スタック

- HTML5 / CSS3 / JavaScript（ES6+）
- Web Speech API（音声読み上げ）
- Web Audio API（効果音生成）
- 外部ライブラリ依存なし
